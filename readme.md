# BigComment for Sublime Text 2

![screenshot](https://raw.github.com/manse/BigComment/master/screenshot.png)

Makes text bigger to displays comment on Minimap.

## Requirements

 * Sublime Text 2

## Installation

 * Download directory to "Packages".

## Usage

 * Windows: Ctrl + Alt + C
 * Mac: Cmd + Option + C
